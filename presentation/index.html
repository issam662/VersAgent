Project Summary

Design and develop a centralized IT infrastructure management platform that provides real-time monitoring, automated compliance enforcement, vulnerability detection, and remote asset management across enterprise workstations via a lightweight desktop agent and modern web dashboard.

Challenges Identified

  • 1- No centralized visibility over PC inventory, software, and network topology across VLANs,
  • 2- Manual compliance checks for mandatory security software (CrowdStrike, Tanium, McAfee),
  • 3- No automated CVE vulnerability detection against installed software versions,
  • 4- Inability to remotely manage, block, or audit workstations without physical access,
  • 5- No VLAN/switch port discovery — network mapping was manual and error-prone

Team Achievements

  • 1- Built a 3-tier architecture: Electron Agent → MSSQL → React Dashboard,
  • 2- Automated real-time PC inventory with heartbeat & installed apps tracking,
  • 3- Implemented CDP/LLDP VLAN discovery using native C# packet sniffer,
  • 4- Created a compliance engine with customizable rules & exception management,
  • 5- Integrated NVD CVE auto-sync for proactive vulnerability scanning

Benefits for the Company

  • 1- 100% automated inventory — zero manual data entry
  • 2- Compliance violations detected in real-time across 500+ machines
  • 3- Reduced IT audit preparation time by 80%
  • 4- Full facility layout visualization with live device status
