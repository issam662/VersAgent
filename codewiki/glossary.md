# Glossary

Key terms used throughout the project.

- **Agent**: An autonomous entity capable of reasoning, planning, and using tools to complete a goal.
- **Tool**: A specific function or API call that an agent can invoke to interact with the world.
- **Reasoning Loop**: The process by which an agent evaluates its current state, plans the next action, and executes it.
- **Orchestrator**: The component (server-side) that manages the lifecycle of an agent's task.
- **Inference**: The process of the model "understanding" and interpreting the user's intent.
- **PFE**: (Project context) The specific project name for this application.
